jsonpp by jmhodges

A fast command line JSON pretty printer.

`jsonpp` is a simple to install and fast to run command line JSON pretty printer. You'll love it.

Dependencies

Nada. Unless you build from source, and then it depends on [go](http://golang.org/doc/install.html).

Install

The binary install is the easiest and most trival way to do it. Go to the Downloads page, find the .tar.bz2 file that matches your computer, download it, and then copy jsonpp to somewhere in your `$PATH`. The source install requires a working [go install](http://golang.org/doc/install.html). Once it's installed run `goinstall github.com/jmhodges/jsonpp`, or run `make install` yourself.
