jsonpp by jmhodges

A fast command line JSON pretty printer.

jsonpp is a simple-to-install and fast-to-run command line JSON pretty printer. You'll love it.

With classic hits like:

curl -s -L http://t.co/tYTq5Pu | jsonpp

And new favorites such as:

jsonpp data/long_malformed.json

Dependencies

Nada. Unless you build from source, and then it depends on go.

Binary Install

Grabbing the binary is the easiest way to do install jsonpp. It's a simple file.

To begin, you'll want to download the zip file that matches your machine:

  • jsonpp for OS X
  • jsonpp for Linux

Then, simply decompress those files and copy the jsonpp binary to somewhere in your $PATH.

To see it in action, pipe some example JSON into it:

curl -s -L http://t.co/tYTq5Pu | jsonpp

Source Install

The source install requires a working install of Go. Once Go is installed run: 

goinstall github.com/jmhodges/jsonpp
or run make install yourself.
